# Hitung BMI

Aplikasi Android untuk menghitung body mass index (BMI) sehingga pengguna dapat mengetahui apakah berat badannya ideal, gemuk atau kurus. Di sini kita belajar bagaimana meminta input dari pengguna, lalu memprosesnya. Kita juga belajar bagaimana memberi feedback jika ternyata input tidak sesuai harapan.
